## Task 03 – Merge Audio with Video using FFmpeg

### Tools Used
- Audacity
- FFmpeg

### Steps
1. Downloaded a 1-minute music clip.
2. Trimmed the audio using Audacity.
3. Merged the audio with the video using FFmpeg.
